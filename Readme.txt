Модуль обмена для решения "Агент Плюс: Мобильная торговля (базовая)" и конфигураций на базе платформы 1С:Предприятие 8.3
версия 2.0.1.4
=================================================================

Модуль обмена представляет собой "внешнюю обработку" (файл с расширением ".epf") для конфигураций на базе платформы 1С:Предприятие 8.3.

Модуль совместим со следующими конфигурациями:

 * «1C:Управление нашей фирмой», версии 1.6.X

Установка и запуск модуля. 
-----------------------------------------------------------------
 1. Скопируйте внешнюю обработку апОбменДанными.epf из данного архива в любое место на диске.
 2. Откройте файл обработки из конфигурации 1С:Предприятия (перечень поддерживаемых конфигураций 1C:Предприятия описан выше).
 3. Следуйте инструкциям в окне обработки.
